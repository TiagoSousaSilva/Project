# Cybersecurity
# Configure the following services in the AWS topology:


1. 

- Create the 'ftpsrv' machine and set up an insecure FTP server.
- Create a user who will access the FTP server. 
- Use Filezilla to access the server.
- Using the Onion server, demonstrate that you can discover the user and password.


2. 

- Centralize the NTP server.
- Remnux obtains time from the Internet, and all other machines get time from Remnux.


3. 

- Check if there is already an MIB browser, and if not, install one on Remnux.
- Activate SNMP (if possible) on the other machines.
- Use the MIB browser to visualize the contents of the other machines.


4. 

- Install a centralized Syslog server on Remnux (pay attention to the ports).


5.

- Configure a site-to-site and remote access VPN.

These are all tasks related to configuring different services and security measures on the AWS platform, with a focus on cybersecurity aspects.
